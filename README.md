# Projet-courbes-de-Beziers

Created by Ahmed Bennasser et Marouane El HAdari
-------------------------------------------------------------

Le projet est composé d'un rapport technique qui contient les details (demonstations , explications des choix , shemas , representaions .... et Resultats)
---------------------------------------------------------------

un repertoire src , qui contient les fichiers .py contenants les implemenations des codes et algorithmes 
(3 fichiers :
***** hermite-q4.py : permet de tracer les Hemites splines avec deux methodes pour le choix des tengents m_k ; (Bessel et Cardinal)
***** Courbure.py  : permet de tracer la courbure d'une fonction ( en utilisant soit Bessel soit Cardinal  ) avec la courbe aussi
***** comparaison.py : ce code permet de tracer la courbure et les courbes par trois methodes et algorithms ; Hermite , Lagrange , SplineC2
)
----------------------------------------------------------------

un repertoire images qui contient des illustrations , des images et des figure permettant de bien comprendre les comparaisons qu'on a fait , aussi des dessins des objets (cat , house , infinie )en utilisant les courbes d'hermite avec Bessel pour le choix des tengentes 
----------------------------------------------------------------

Ce projet est un travail collaboratif pour objectif de comparer les performances des courbes (Hermite , Bezier , SplineC2 , Lagrange ) mais aussi pour voir comment le choix des tengentes (Bessel , Cardinal) peut affecter les performances et la stabilité
